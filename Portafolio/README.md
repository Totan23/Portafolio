# Mi Web

Sitio web estático simple — HTML, CSS y JavaScript vanilla.

## 🚀 Deploy rápido

### Netlify (drag & drop)
1. Ve a [netlify.com](https://netlify.com) → "Add new site" → "Deploy manually"
2. Arrastra la carpeta `mi-web/` al área de drop
3. ¡Listo! Obtienes una URL pública al instante.

### GitHub Pages
1. Sube el proyecto a un repositorio de GitHub
2. Ve a Settings → Pages → Source: `main` branch / `root`
3. GitHub genera la URL automáticamente.

### Vercel (recomendado)
1. Instala Vercel CLI: `npm i -g vercel`
2. Ejecuta: `vercel` dentro de la carpeta
3. Sigue los pasos del CLI — deploy en segundos.
